# ButterBackup – Backups so geschmeidig wie Butter!

**Inkrementelle Vollbackups dank BtrFS**

---

📖 **Ausführliche Dokumentation:**
**[🇩🇪 Deutsch (README.de.md)](README.de.md)** | [🇬🇧 English (README.en.md)](README.en.md)

---

ButterBackup ist ein Kommandozeilenwerkzeug, das das Anlegen von
Sicherungskopien auf verschlüsselten externen Datenträgern so einfach wie
möglich macht. Es genügt, die Festplatte anzuschließen und das Programm zu
starten – ButterBackup erledigt den Rest.

## Schnellstart

    pipx install butter-backup
    butter-backup backup --config ~/.config/butter-backup.cfg
